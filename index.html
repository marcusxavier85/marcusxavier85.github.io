Pokedex Collection App

This is was our first major project that we took on. It uses PHP for the backend to hook up to a database in Sequel Pro. The PHP then in turn displays the database in a table format. This App also allows you to add new Pokemon to the Pokedex. Which will take the inputted data from the form and submit it into the database.

Jarver's Crypt Maze Game

Jarver's Crypt was the first project we did which incorporated JavaScript into our project. We used it to animate the moving obstacles, set timers, notify when the player hit something and someother things. A lot of this was done through the DOM with a lot of query selectors. With regards to animation this project uses the new animate function so there is no jQuery involved.

Ecommerce Store

This was a group project we worked on. This was the first major project where we used OOP. We focused on adhering to the SOLID principles and backend PHP. This project is linked to a database that is hooked up the frontend via PHP which then displays the products and each character, description, price etc associated with them.

Academy Portal

With this project we continued working on an application that had been built by previous developers. It was built on the Slim framework and was the first project where we used MVC architecture. This was a full stack build so HTML, CSS, JavaScript, MySql and PHP were all used.
